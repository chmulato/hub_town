# Hub

Um projeto full-stack moderno com front-end React e back-end Node.js.

## Descrição

Este projeto é composto por duas partes principais:
- **Front-end**: Interface do usuário desenvolvida em React (JSX)
- **Back-end**: API servidor desenvolvida em Node.js

## Estrutura do Projeto

```
hub/
├── README.md
├── back-end/
│   └── server.js
├── doc/
└── front-end/
    └── front.jsx
```

## Pré-requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn

## Instalação

### Back-end
```bash
cd back-end
npm install
```

### Front-end
```bash
cd front-end
npm install
```

## Execução

### Executar o Back-end
```bash
cd back-end
npm start
```

### Executar o Front-end
```bash
cd front-end
npm start
```

## Funcionalidades

- API RESTful no back-end
- Interface responsiva no front-end
- Comunicação entre front-end e back-end

## Tecnologias Utilizadas

### Back-end
- Node.js
- Express.js (presumido)

### Front-end
- React
- JSX
